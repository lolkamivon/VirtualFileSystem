# 📂 Віртуальна файлова система (Virtual File System)

Десктопний додаток для симуляції роботи віртуальної файлової системи, розроблений мовою C++ з використанням фреймворку Qt. Проєкт реалізує класичні алгоритми розподілу пам'яті, контроль доступу та логування подій. 

Розроблено в рамках курсового проєкту з інженерії програмного забезпечення.

## ✨ Основні можливості

* **Емуляція жорсткого диска:** Усі дані та структури файлової системи зберігаються у єдиному бінарному файлі-контейнері (`virtual_disk.bin`) фіксованого розміру (200 МБ) з розбиттям на 512-байтові сектори.
* **Суміжне виділення пам'яті:** Реалізовано алгоритм алокації простору "Перший підходящий" (First Fit).
* **Дворівнева ієрархія:** Підтримка кореневого каталогу (для створення папок) та каталогів користувачів (для зберігання файлів).
* **Рольова модель безпеки (RBAC):** Розмежування прав доступу. Адміністратор має повний контроль над системою, звичайний користувач може керувати лише власними файлами.
* **Повний набір операцій (CRUD):** Форматування диска, створення, читання, видалення (відкладене/логічне), перейменування, копіювання та переміщення об'єктів.
* **Безперервне логування (Аудит):** Усі дії користувачів та системні транзакції автоматично записуються у файл `vfs_system.log` із зазначенням часових міток.
* **Графічний інтерфейс (GUI):** Зручна взаємодія через віконний інтерфейс на базі Qt Widgets без необхідності використання терміналу.

## 🛠 Технологічний стек

* **Мова програмування:** C++ (стандарт C++17)
* **Графічний фреймворк:** Qt 6 (Qt Widgets)
* **Система збирання:** CMake
* **ОС для розробки та тестування:** Linux
